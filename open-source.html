Juji Open Source Software

  1. Datalevin: a fast Datalog database.
  2. Editscript: a library to diff and patch Clojure/Clojurescript data structures.
  3. Gibberish-detector: a library to detect gibberish user input.
  4. Juji cli-client: a node.js command line client that demonstrates the use of Juji API.
  5. Juji Speak: a GUI client that uses Juji API along with voice API (STT and TTS) to power a Juji cognitive AI assistant with voice.
